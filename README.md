## 📌 Project Overview

DeepMineRisk is an AI-powered mining process monitoring system that predicts future mining process conditions using Long Short-Term Memory (LSTM) deep learning. The model analyzes multivariate sensor data to estimate operational risk levels and assist in proactive safety monitoring.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- LSTM Time-Series Prediction
- Risk Score Prediction
- Safe / Warning / Critical Classification
- Training & Validation Visualization
- Model Saving
- Prediction Report Generation

## 📈 Model Architecture

- LSTM (32 Units)
- Dropout (0.2)
- LSTM (16 Units)
- Dense Layer (ReLU)
- Output Layer

Optimizer: Adam

Loss Function: Mean Squared Error (MSE)
